fn aggregate(&self, group: &str, tasks: Vec<Task>) -> Result<Task>

聚合给定组中的任务，返回聚合后的新任务 Aggregates the given tasks in a group and returns a new aggregated task

§Arguments
  • group - 组名 / Group name
  • tasks - 要聚合的任务列表 / List of tasks to aggregate
§Returns

聚合后的新任务。队列选项会被忽略，聚合任务总是入队到组所属的队列 The aggregated new task. Queue option will be ignored, aggregated task always enqueues to the group’s queue
