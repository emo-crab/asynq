Periodic Task Manager - 管理周期性任务 Periodic Task Manager - manages periodic tasks

对应 Go asynq 的 PeriodicTaskManager 组件 Corresponds to Go asynq’s PeriodicTaskManager component

此组件负责： This component is responsible for:

  1. 从 PeriodicTaskConfigProvider 获取任务配置
  2. Get task configurations from PeriodicTaskConfigProvider
  3. 通过 Scheduler 注册和注销周期性任务
  4. Register and unregister periodic tasks via Scheduler
  5. 根据 diff 结果动态同步任务
  6. Dynamically sync tasks based on diff results
