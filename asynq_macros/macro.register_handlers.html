Macro register_handlers 

Source 
register_handlers!() { /* proc-macro */ }
Expand description

Macro for automatically registering synchronous handlers with ServeMux

This macro collects all handlers defined in the current scope and registers them with a ServeMux instance.

§Examples

ⓘ
use asynq::register_handlers;
use asynq::serve_mux::ServeMux;

let mut mux = ServeMux::new();
register_handlers!(mux, handle_email, handle_image_resize);
